# Proyectos-web
Desarrollo de  mini proyectos web interactivos. 

Este repositorio está constituido por mini proyectos web a los que se pueden acceder desde la página principal.
Los proyectos  creados son :
- Test de mecanografía en la que los usuarios pueden interactuar mediante el teclado y se visualiza por pantalla las teclas y palabras que el usuario tiene que escribir.
- Juego de tablero en el que usuario puede arrastrar unas fichas hasta tenerlas ordenadas.
- Mapa de las provincias de Andalucía en la que al clicar en ella se puede visualizar información de la provincia como la gastronomía o alguna característica clave.
- Piano interactivo en la al pulsar las teclas numéricas se puede escuchar una nota del piano.
- Pizarra interactiva (Canvas).
